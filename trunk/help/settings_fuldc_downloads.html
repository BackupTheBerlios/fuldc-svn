Downloads

Skiplist

A list over filenames that should not be downloaded, the filenames are separated by |. It is possible to specify wildcards such as * ? and [].

Static "download to" directories

The paths specified here will show up in the download to menus when searching and browsing users lists.

Add
Opens up a dialog that lets you choose the path, after pressing ok it will ask about a name for that path that should be displayed in the menus.
Remove
removes the selected entry in the list.

max sources for match queue

The number entered here specifies when fulDC should stop automatically downloading and matching users queues. A value of 5 means that a download has to have 5 sources before it should stop.
