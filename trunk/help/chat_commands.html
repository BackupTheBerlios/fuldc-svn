Chat Commands

Main Chat

/join <hub-ip>
.
/clear
Clears the main chat window of all text.
/ts
Toggles the showing of time stamps on new chat messages.  It does not add timestamps to already received chat lines.
/showjoins
Toggles the displaying of users joining the hub.  This only takes effect for freshly-arriving users.
/favshowjoins
Toggles the displaying of favorite user joining the hub.
/close
Closes the hub window.
/userlist
Toggles visibility of the list of users for the current hub.
/connection
Displays the IP and port that DC++ will tell users to connect to.  If either the IP or port is not correct, you will be unable to download from some users.  This is only useful in active mode - the IP and port are unused in passive mode.
/favorite
/fav
Adds the current hub (along with your nickname and password, if used) to the list of Favorite Hubs.
/pm <user> [message]
Opens a private message window to the user, and optionally sends the message, if one was specified.
/getlist <user>
Adds the user's list to the Download Queue.
/grant <user>
Grants a slot to the user specified. If strip isp is activated use tab to select the correct user in the userlist.
/topic
Displays the current topic in the chat. Use this if the topic is too long to fit in the window title.
/ctopic
Opens links in the current topic in your default web browser.
/popups
Toggles use of popups. This controls the use of popups of the entire application
/log
Toggles logging of the current hub
/lastseen <user>
Tries to find when the user last said something in the chat and highlights it if a match is found
/dns <dns or ip>
If an IP is given it returns the host name for that IP. If a DNS name is given it returns the IP.

Private Message

/getlist
Adds the current user's list to the Download Queue.
/clear
Clears the current Private Message window.
/grant
Grants the remote user a slot.  Once they connect, or if they don't connect in 10 minutes, the granted slot is removed.
/close
Closes the current Private Message window.
/favorite
/fav
Adds the current user to the list of Favorite Users.
/mute
Toggles sounds for the current private message
/unmute
Enables sound for the current private message. This is an old setting and will probably be removed in later versions.
/pop
Toggles use of popups for the current private message

Both

/refresh <directory>
Manually refreshes DC++'s share list by going through the shared directories and adding new folders and files.  If no directory is specified the whole share is refreshed, if a virtual directory is specified all physical directories with that virtual name is refreshed and if a physical directory is specified it will be refreshed. DC++ automatically refreshes once an hour by default, and also refreshes after the list of shared directories is changed.
/refreshi
Refreshes only directories marked as incoming. (the little checkbox to the left of directories in settings dialog)
/slots <number>
Sets the current number of upload slots to the number you specify.  If this is less than the current number of slots, no downloads are cancelled.
/search <string>
Opens a new search window with the specified search string.  It does not automatically send the search.
/dc++
Sends a random DC++ advertising message to the chat, including a link to the DC++ homepage and the version number.
/away [message]
Sets Away status.  New private message windows will be responded to, once, with the message you specified, or the default away message configured in the Appearance settings page.
/back
Un-sets Away status.
/g <search string>
Launches your default web browser to the Google search engine with the specified search.
/i <imdb query>
Launches your default web browser to the Internet Movie Database (imdb) with the specified query.
/t <search string>
Launches your default web broswer to tv tome with the specified search.
/rebuild
Rebuilds the HashIndex.xml and HashData.dat files, removing entries to files that are no longer shared, or old hashes for files that have since changed.  This runs in the main DC++ thread, so the interface will freeze until the rebuild is finished.
/showlog <status, system, downloads, uploads>
If no parameter is specified, it launches the log for the hub or private chat with the associated application in Windows. If one of the parameters is specified it opens that log file. The status log is available only in the hub frame.
/lastlog
Opens a window displaying the last 100 lines that matches a highlight with lastlog enabled.
/me <message>
Doesn't really do anything, simply sends <nick> /me message as a chat message. This is mostly used to talk about yourself in third person.
/share <path>
Tries to add path to the share, the virtual name will be automatically chosen.
/unshare <path>
Removes path from the share.
/fuldc
Displays the url to fulDC's website and the version you're using.
/fuptime
Sends a chat message displaying how long fulDC has been running.
/uptime
Sends a chat message displaying how long since you started your computer.
/df
Sends a chat message displaying your total and free HDD space
