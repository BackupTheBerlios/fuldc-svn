Sharing

Skiplist

A list over filenames that should not be shared, the filenames are separated by |. It is possible to specify wildcards such as * ? and [].

Minislots

file extensions
This is a list of files that should be allowed to be downloaded using a minislot. The list is separated by | and supports wildcards.
Size
This is the maximum size in KiB a file should be to use a minislot.

Refresh Options

Automatically refresh incoming
If this is enabled you can specify a time in minutes between refreshes of the directories marked as incoming.
Automatically refresh share
If this is enabled you can specify a time in minutes between refreshes of the whole share.
