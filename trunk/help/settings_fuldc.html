fulDC

Boolean Options

Expand Queue
This controls how the download queue should look when the windows is opened. If enabled the queue will show all folders expanded otherwise only the root folders will be expanded.
Strip ISP(public hubs only, see favorite hub settings)
Controls how nicks should be displayed, if enabled leading brackets will be removed from the nick. e.g [ISP]nick will be displayed as nick
Strip ISP in private messages.
Strip ISP tag in private messages.
Set high priority no samples and subs when enqueued
If enabled samples and subs will get a higher priority than other files to make sure they're downloaded first.
Create new mainchat log every midnight
This option splits the mainchat logs in to smaller log files. A new file is created every day and named after the current date.
Confirm hub exit
If enabled a dialog box will pop up asking if you really want to close the hub window.
confirm removal of files from queue
If enabled a dialog box will pop up asking if you really want to remove the files.
play custom sounds on PM
With this enabled add PM.wav and newPM.wav to your fuldc directory to have fuldc play those sounds when receiving private messages instead of the default sounds.
Remove topic (everywhere except hub window title)
Removes everything after the last - found in the hub name. Enable this to get a cleaner look everywhere except hub window title.
Toggle window when selecting an active tab
If this is activated, clicking on an active tab will deactivate that window and activate the next window in the tab order.
Drop connections from own nick(stupid clients)
If you're seeing connection attempts from your own nick and it bothers you, activate this function to drop those connections. If logging of system messages is enabled, a log entry will be created in system.log.
Flash window on PM
Flash window when a private message is received.
Flash window on PM only if it's from a new user
Only display flash the window when the PM window is created for that user.
Ignore TTH inconsistency
If you do not want to lose the slot when encountering a TTH inconsistency enable this and the file will continue downloading. The file will probably be broken but at least you get to keep the slot and it's easy to verify with sfv.

Web Shortcuts

Web shortcuts are used to get easy access to websites, usually sites where you can search for information about a specific subject.

Add
Used to add a new web shortcut to the list. This brings up a dialog with a few fields. Name is what is displayed in menus where web shortcuts can be used. Key is used in hub and pm windows, /<key> <search term> to open the specified url. Url is the url that should be opened when a web shortcut is used, specify %s in the url where the search term should be inserted. Clean search string is used to remove info from the search string that's not necessary, for example "Movie.Name.2003.DVDRip.XviD-ABC" will become "Movie Name"
Properties
Opens the same dialog as Add with the values for the selected web shortcut filled in.
Remove
Removes the selected web shortcut.
