Highlight

Add
Opens a dialog that lets you configure a new highlight.
Remove
Removes the selected item from the list.
Change
Opens a dialog that lets you change the settings of the selected highlight.
Move Up
Moves the selected item up one step in the list. Highlights are run in the order they're stored in the list so highlights at the bottom might write over colors added by highlights at the top of the list.
Move Down
Moves the selected item down one step in the list.
Presets
These are a set of preconfigured highlights for common words/phrases. Note they are configured to match the default background and text color, which is white background and black text.

Settings

Text Styles

Bold
Bolds the text.
Italic
Makes the text italic
Underline
Underlines the text
Strikeout
Adds a line through the text.
Background Color
Check the box and select a color to be used to highlight the background around the text.
Text Color
Check the box and select a color to be used to highlight the text.

Match Options

Include Nick
Means that it should start searching from the beginning of the line. Normally a search is started after the nick of the user.
Whole Line
Means that any color or text options should be applied to the whole line.
Case Sensitive
Means that it should be case sensitive when searching for a match.
Entire Word
Means that any color or text options should be applied to the entire word, if the highlight just matched a part of the word.
Match Type
Begins means that the word specified should match the beginning of a word in the text.
Contains means that the word specified should match anything.
Ends means that the word specified should match the end of a word in the text.
Equals means that the word specified should be an exact match to a word in the text.

Actions

Display popup
Displays a popup, check popups page for options.
Add to history
Add the matching line to history. The last 100 matching lines are stored and can be viewed by typing /history in a hub window. There is a separate history saved for each hub.
Flash Window
Will flash the the taskbar entry if it matches, only if the windows isn't active.
Change Tabcolor
Will change the tabcolor, check tabs page to see how to specify the color
Play Sound
Plays a sound specified by pressing the "Select Sound File" button and choosing a .wav file.

Text to match

Specify which word should be searched for, can be a letter, part of a word or a whole sentence if you like.

Regexps

It is possible to start the line with $Re: to tell fulDC that the text is a regexp. Do not use regexps if you don't know how to use them, most stuff is possible to do without them. starting the line with $Re: disables a few of the controls such as include nick, whole line, entire word and case sensitive.

Predefined variables

$users$
Use this to highlight all nicks in the chat
$ts$
Use this to highlight timestamps.
$mynick$
Use this if you wan't to highlight your nick but you're using different nicks in different hubs.
