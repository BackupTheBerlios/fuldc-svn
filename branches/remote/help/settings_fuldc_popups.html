Popups

It is possible to change the appearance of the popups by placing a .bmp named popup.bmp in /icons . If no bitmap is found it defaults to the standard windows window color.

Boolean Options

Only show popups if away
Self explaining. Only show popups if away mode is activated.
Only show popups if minimized to tray
Self explaining. Only show popups if fuldc is minimized to the tray.
Show popup on PM
Display a popup when a private message is received.
Show popup on PM only if it's from a new user
Only display a popup when the PM window is created for that user.
Show popup on hub status messages such as disconnected
Self explaning. Show a popup if you're disconnected from the hub.
Automatically close popup windows
Close popup windows when a certain amount of time has passed.
Activate window when clicking popup
Activate the window responsible for the popup when it's clicked.
Don't display popups for active window
Don't display any popups for the window that currently has focus.

Popup time in seconds

Determines how long a popup should be displayed if Automatically close popup windows is enabled.

Max message length

Determines how many characters should be displayed in the popup.

Appearance

Font
Specifies the font that should be used in the popups.
Text color
Specifies the text color that should be used in the popups.
